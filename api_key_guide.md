# API 키 발급 및 설정 가이드

본 가이드는 앱 내의 **미세먼지(Air Korea)** 및 **기상 정보(KMA)** 데이터를 활성화하기 위해 필요한 전용 API 키 발급 방법을 설명합니다.

---

## 1. 에어코리아(Air Korea) API 키 발급 방법

1. **공공데이터포털 홈페이지**에 접속 및 로그인합니다. ([data.go.kr](https://www.data.go.kr))
2. 검색창에 **'한국환경공단_에어코리아_대기오염정보'**를 검색합니다.
3. 결과에서 해당 항목을 선택한 후 **[활용신청]** 버튼을 클릭합니다.
4. 신청 페이지의 '활용 목적' 등에 간단한 내용을 기입하고 승인합니다. (대부분 즉시 승인됩니다.)
5. **마이페이지 > 일반 인증키** 섹션에서 **인증키(Decoding)** 항목을 복사합니다.

> [!IMPORTANT]
> **중요**: 파이썬 `requests` 라이브러리를 사용할 때는 반드시 **'Decoding'** 키를 사용해야 오류가 발생하지 않습니다.

---

## 2. 기상청(KMA) API 키 발급 방법 (자외선 지수 등)

1. 동일한 공공데이터포털에서 **'기상청_생활기상지수 조회서비스'**를 검색합니다.
2. 위와 동일한 방법으로 활용 신청을 하고 **인증키(Decoding)**를 복사합니다.

---

## 3. 앱 코드에 적용하기

발급받은 키를 실시간으로 적용하려면 `main.py` 파일 상단의 설정 부분을 다음과 같이 수정해 주세요.

```python
# main.py 상단 수정
KMA_API_KEY = "발급받은_기상청_디코딩_키"
AIR_KOREA_API_KEY = "발급받은_에어코리아_디코딩_키"
```

> [!TIP]
> 키를 입력한 후 앱을 다시 빌드하거나 실행하면 미세먼지와 자외선 카드가 자동으로 활성화됩니다.
